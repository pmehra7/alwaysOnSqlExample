# alwaysOnSqlExample
An example of how to create an app to query the AlwaysOnSQL service 

## Build
`mvn clean package`

## Run 
`java -cp alwaysOnSqlQueryApp-1.0-SNAPSHOT-jar-with-dependencies.jar
 com.datastax.examples.ClusterConnect`
